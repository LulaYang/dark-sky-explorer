# Dark Sky Explorer

An interactive web page exploring International Dark Sky locations across the United States.

## Features
- Click buttons to switch between dark sky locations
- View photos and information about each location
- Interactive Mapbox map showing all locations

## Built With
- HTML, CSS, JavaScript
- Mapbox GL JS

## Locations
- Sedona, Arizona
- Flagstaff, Arizona
- Beverly Shores, Indiana
- Old Snowmass, Colorado
